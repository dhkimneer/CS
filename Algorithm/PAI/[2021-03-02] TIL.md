## Q61. Largest Number (Leetcode Q179)

<br>

항목들을 조합하여 만들 수 있는 가장 큰 수를 출력하라.

```python
[예제1]
입력: [10, 2]
출력: "210"
    
[예제2]
입력: [3, 30, 34, 5, 9]
출력: "9534330"
```

<br>

#### [풀이1. 삽입 정렬]

<br>

```python
from typing import List


class Solution:
    # 문제에 적합한 비교 함수
    @staticmethod
    def to_swap(n1: int, n2: int) -> bool:
        return str(n1) + str(n2) < str(n2) + str(n1)
    
    # 삽입 정렬 구현
    def largest_number(self, nums: List[int]) -> str:
        i = 1
        while i < len(nums):
            j = i
            while j > 0 and self.to_swap(nums[j - 1], nums[j]):
                nums[j], nums[j - 1] = nums[j - 1], nums[j]
                j -= 1
            i += 1
            
        return str(int(''.join(map(str, nums))))
```

## Q62. Valid Anagram (Leetcode Q242)

<br>

t가 s의 애너그램인지 판별하라.

```python
[예제 1]
입력: s = "anagram", t = "nagaram"
출력: True
    
[예제 2]
입력: s = "rat", t = "car"
출력: False
```

<br>

#### [풀이1. 정렬을 이용한 비교]

<br>

```python
class Solution:
    def is_anagram(self, s: str, t: str) -> bool:
        return sorted(s) == sorted(t)  # 새로운 정렬 리스트 반환
```

<br>

- route

```python
[예제 2]
입력: s = "rat", t = "car"
출력: False
    
    
1. sorted("rat")
>>> ["a", "r", "t"]

2. sorted("car")
>>> ["a", "c", "r"]

["a", "r", "t"] != ["a", "c", "r"]  # return False
```

## Q63. Sort Colors (Leetcode Q75)

<br>

빨간색을 0, 흰색을 1, 파란색을 2라 할 때 순서대로 인접하는 제자리 정렬을 수행하라.

```python
입력: [2, 0, 2, 1, 1, 0]
출력: [0, 0, 1, 1, 2, 2]
```

<br>

#### [풀이1. 네덜란드 국기 문제를 응용한 풀이]

<br>

```python
from typing import List


class Solution:
    def sort_colors(self, nums: List[int]) -> None:
        red, white, blue = 0, 0, len(nums)
        
        while white < blue:
            if nums[white] < 1:
                nums[red], nums[white] = nums[white], nums[red]
                white += 1
                red += 1
            elif nums[white] > 1:
                blue -= 1
                nums[white], nums[blue] = nums[blue], nums[white]
            else:
                white += 1
```

## Q64. K Closest Points to Origin (Leetcode Q973)

<br>

평면상에 points 목록이 있을 때, 원점 (0, 0)에서 K번 가까운 점 목록을 순서대로 출력하라. 평면상 두 점의 거리는 유클리드 거리로 한다.

```python
[예제 1]
입력: points = [(1, 3), (-2, 2)], K = 1
출력: [(-2, 2)]
    
[예제 2]
입력: points = [(3, 3), (5, -1), (-2, 4)], K = 2
출력: [(3, 3), (-2, 4)]
```

<br>

#### [풀이1. 유클리드 거리의 우선순위 큐 순서]

<br>

```python
import heapq
from typing import List


class Solution:
    def k_closest(self, pointsL List[List[int]], K: int) -> List[List[int]]:
        heap = []
        for (x, y) in points:
            dist = x ** 2 + y ** 2
            heapq.heappush(heap, (dist, x, y))
            
        result = []
        for _ in range(K):
            (dist, x, y) = heapq.heappop(heap)
            result.append((x, y))
        return result
```
