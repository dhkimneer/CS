## Q45. Invert Binary Tree (Leetcode Q226)

<br>

입력: [4, 2, 7, 1, 3, 6, 9]

출력: [4, 7, 2, 9, 6, 3, 1]

<br>

#### [풀이1. 파이썬다운 방식]

<br>

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
        
        
class Solution:
    def invert_tree(self, root: TreeNode) -> TreeNode:
        if root:
            root.left, root.right = self.invert_tree(root.right), self.invert_tree(root.left)
            return root
        return None
```

#### [풀이2. 반복 구조로 BFS]

<br>

```python
import collections


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
        
        
class Solution:
    def invert_tree(self, root: TreeNode) -> TreeNode:
        queue = collections.deque([root])
        
        while queue:
            node = queue.popleft()
            # 부모 노드부터 하향식 스왑
            if node:
                node.left, node.right = node.right, node.left
                
                queue.append(node.left)
                queue.append(node.right)
                
        return root
```

#### [풀이3. 반복 구조로 DFS]

<br>

```python
import collections


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
      
    
class Solution:
    def invert_tree(self, root: TreeNode) -> TreeNode:
        stack = collections.deque([root])
        
        while stack:
            node = stack.pop()  # DFS 탐색 시 pop()으로 마지막 값 추출
            # 부모 노드부터 하향식 스왑
            if node:
                node.left, node.right = node.right, node.left
                
                stack.append(node.left)
                stack.append(node.right)
                
        return root
```

#### [풀이4. 반복 구조로 DFS 후위 순회]

<br>

```python
import collections


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
        

class Solution:
    def invert_tree(self, root: TreeNode) -> TreeNode:
        stack = collections.deque([root])
        
        while stack:
            node = stack.pop()
            
            if node:
                stack.append(node.left)
                stack.append(node.right)
                
                node.left, node.right = node.right, node.left  # 후위 순회
                
        return root
```

## Q46. Merge Two Binary Trees (Leetcode Q617)

<br>

두 이진 트리를 병합하라. 중복되는 노드는 값을 합산한다.

입력) 

트리1: [1, 3, 2, 5] / 트리2: [2, 1, 3, null, 4, null, 7]

출력)

[3, 4, 5, 5, 4, null, 7]

<br>

#### [풀이1. 재귀 탐색]

<br>

```python
import collections


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
        

class Solution:
    def merge_trees(self, t1: TreeNode, t2: TreeNode) -> TreeNode:
        if t1 and t2:
            node = TreeNode(t1.val + t2.val)
            node.left = self.merge_trees(t1.left, t2.left)
            node.right = self.merge_trees(t1.right, t2.right)
            
            return node
        else:
            return t1 or t2
        	# 둘 중 하나만 참일 경우 참인 값을 반환
```

## Q47. Serialize and Deserialize Binary Tree (Leetcode Q297)

<br>

이진 트리를 배열로 직렬화하고, 반대로 역직렬화하는 기능을 구현하라. 즉 다음과 같은 트리는

[1, 2, 3, null, null, 4, 5] 형태로 직렬화할 수 있을 것이다.

<br>

- 직렬화: 데이터 구조를 파일이나 디스크에 저장하기 위해 물리적 형태로 바꾸는 것

<br>

#### [풀이1. 직렬화 & 역직렬화 구현]

<br>

```python
import collections


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
        
        
class Codec:
    # 직렬화
    def serialize(self, root: TreeNode) -> str:
        queue = collections.deque([root])
        result = ['#']  # null 대신 '#'로 표현 (python null인 None을 문자열로 만들 수 없기 때문)
        # 트리 BFS 직렬화
        while queue:
            node = queue.popleft()
            if node:
                queue.append(node.left)
                queue.append(node.right)
                
                result.append(str(node.val))
            else:
                result.append('#')
        return ' '.join(result)
    
    # 역직렬화
    def deserialize(self, data: str) -> TreeNode:
        # 예외 처리
        if data == '# #':
            return None
        
        nodes = data.split()
        
        root = TreeNode(int(nodes[1]))
        queue = collections.deque([root])
        index = 2
        # 빠른 런너처럼 자식 노드 결과 먼저 확인 후 큐 삽입
        while queue:
            node = queue.popleft()
            if nodes[index] is not '#':
                node.left = TreeNode(int(nodes[index]))
                queue.append(node.left)
            index += 1
            
            if nodes[index] is not '#':
                node.right = TreeNode(int(nodes[index]))
                queue.append(node.right)
            index += 1
        return root
```
